# The Delicious

A simple restaurant website that showcases a menu, gallery, and contact details.  
Built using HTML and CSS.

## Preview

The site includes:

- A fixed navigation bar
- Hero section with a call-to-action button
- About section
- Menu with categories and prices
- Image gallery
- Contact information
- Footer

## Project Structure

project-folder/
│
├── index.html
├── readme.md
├── style.css
└── images
    ├── margherita-pizza.png
    ├── hyderabadi-chicken-biryani.png
    └── white-sauce-pasta.png


## How to Run

1. Download or clone the project.
2. Make sure the `style.css` and `images` folder are in the same directory as `index.html`.
3. Open `index.html` in your browser.

## Features

- Responsive layout
- Organized menu categories
- Clean and simple UI
- Easy to customize

## Customization

We can update:

- Text content inside `index.html`
- Colors, spacing and fonts in `style.css`
- Images inside the `images` folder
